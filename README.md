# 踩地雷遊戲 (Minesweeper)

一個使用純HTML/CSS/JavaScript實現的經典踩地雷遊戲，具有完整的遊戲功能和音效系統。

## 功能說明

### 1. 遊戲核心功能
- **完整的掃雷遊戲邏輯**：放置地雷、計算相鄰地雷數、揭示格子
- **左鍵點擊**：揭示格子
- **右鍵點擊**：標記旗幟或問號
- **自動展開**：點擊空白格子時自動展開相鄰區域

### 2. 難度選擇
- **初級**：9×9 格子，10 個地雷
- **高級**：16×16 格子，40 個地雷
- **自訂難度**：可自定義寬度(8-30)、高度(8-24)、地雷數和倒數秒數

### 3. 計時器系統
- **倒數計時模式**：從設定時間開始倒數，時間到則遊戲結束
- **正數計時模式**：從0開始計算遊戲時間
- **心跳音效**：倒數最後30秒開始心跳聲
- **滴答聲**：倒數最後10秒開始滴答聲

### 4. 音效系統 (Web Audio API)
- 心跳音效（倒數最後30秒）
- 滴答聲（倒數最後10秒）
- 爆炸音效（踩到地雷時）
- 勝利音效（播放一串上升音階）

### 5. 遊戲界面
- 經典Windows風格的掃雷界面
- 表情按鈕：點擊可重新開始遊戲
- 地雷計數器：顯示剩餘地雷數（扣除已標記旗幟）
- 計時器顯示

### 6. 遊戲狀態管理
- **勝利檢測**：揭示所有非地雷格子後獲勝
- **失敗處理**：踩到地雷時顯示爆炸動畫和遊戲結束畫面
- **重新開始**：可隨時重新開始遊戲
- **菜單界面**：用於選擇難度

### 7. 視覺效果
- 爆炸時網格震動動畫
- 地雷揭示時的閃爍效果
- 勝利時表情動畫
- 響應式設計，支援移動設備

### 8. 其他功能
- 首次點擊保證安全（不會踩到地雷）
- 遊戲暫停（切換計時模式時重置計時器）
